ยินดีต้อนรับสู่ NumberNiceIC!

Landing Page นี้แสดงความสามารถของ Fiber ในการใช้ Go HTML Template โดยแยกส่วนประกอบเป็น Layout และ Partial ได้อย่างสมบูรณ์

โครงสร้างที่ใช้

  • **Layout:** `layouts/main.html`
  • **Partial (ส่วนย่อย):** `partials/header.html` และ `partials/footer.html`
  • **Content:** `pages/index.html`

คุณสามารถเพิ่ม CSS Framework เช่น Tailwind หรือ Bootstrap ได้อย่างง่ายดายเพื่อความสวยงามยิ่งขึ้น
